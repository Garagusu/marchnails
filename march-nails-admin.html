🗄️ Supabase — Cross-Device Bookings
Connect a free Supabase database so bookings appear on ALL devices — PC, iPad, iPhone.
Setup (free, 2 minutes): 1. Create free Supabase account
2. New Project → go to SQL Editor → paste and run:
CREATE TABLE bookings (id text primary key, data jsonb, created_at timestamptz default now());
ALTER TABLE bookings ENABLE ROW LEVEL SECURITY;
CREATE POLICY "pub_insert" ON bookings FOR INSERT WITH CHECK (true);
CREATE POLICY "pub_select" ON bookings FOR SELECT USING (true);
CREATE POLICY "pub_update" ON bookings FOR UPDATE USING (true); 3. Settings → API → copy Project URL and anon key below
